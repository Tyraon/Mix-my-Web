Anleitung

Mix my Web ist eine Webanwendung im Browser und benötigt keine weiteren Plugins wie einen Flashplayer.

Warum Mix my Web

Mix my Web ist eine Weboberfläche welche in jedem aktuellen Desktopbrowser genutzt werden kann. Somit besteht eine kompatibilität zu den verschiedenen Betriebssystemen. Es ermöglicht eine einfache Wartung und Aktuallisierung ohne, dass der Benutzer selbst eine Aktuallisierung oder ein Update vornehmen muss. Durch die Einfachheit der Scriptsprachen besteht auch nur eine geringe Auslastung am Client selbst, bietet aber fast den gleichen Komfort wie eine Desktopanwendung.
Zudem ist und bleibt Mix my Web kostenfrei.

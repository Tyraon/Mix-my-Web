Anleitung

Mix my Web ist eine Webanwendung im Browser und benötigt keine weiteren Plugins wie einen Flashplayer.

Warum Mix my Web

Mix my Web ist eine Weboberfläche welche in jedem aktuellen Desktopbrowser genutzt werden kann. Somit besteht eine kompatibilität zu den verschiedenen Betriebssystemen. Es ermöglicht eine einfache Wartung und Aktuallisierung ohne, dass der Benutzer selbst eine Aktuallisierung oder ein Update vornehmen muss. Durch die Einfachheit der Scriptsprachen besteht auch nur eine geringe Auslastung am Client selbst, bietet aber fast den gleichen Komfort wie eine Desktopanwendung.
Zudem ist und bleibt Mix my Web für den Nutzer kostenfrei.

Bedienung

Nach dem öffnen von Mix my Web steht die Oberfläche sofort zur Verfügung. Als erstes fragt der Webbrowser nach dem Zugriff auf das Mikrofon, welcher notwendig ist um dieses in Mix my Web nutzen zu können. Jetzt können die einzelnen Tracks in die Playlist geladen werden. Dazu klicke einfach im Playlistfenster (unten-links) auf den '+' Button. Es öffnet sich nun ein Fenster in dem man sich den ersten Track auswählen und einladen kann. So kann die Playlist nach und nach befüllt werden.
Um nun einen Track abzuspielen klicke einfach auf den '<' Button neben dem Track um diesen in Deck 1 einzuladen oder den '>' Button um den Track in Deck 2 einzuladen. Über einen Klick auf den Playbutton wird der Track nun abgespielt.

Jingles (oder Effektsounds) können im Fenster 'Jingles' eingeladen werden. Hier ist bei der Auswahl auch eine Mehrfachauswahl möglich. Alle eingeladenen Jingles werden dann im Fenster angezeigt und können über den jeweiligen Playbutton direkt und parallel zum laufenden Track abgespielt werden.

Im mittleren Bereich finden wir verschiedene Funktionen. Von Links nach Rechts haben wir als erstes die Regler für die Höhen und die Tiefen, welche auf den Masteroutpu gerichtet sind. Daneben befindet sich der Crossfader, mit dem zwischen den Decks gewechselt werden kann. Als letztes haben wir hier noch das Mikrofon. Dies ist standartmäßig deaktiviert kann aber mit einem Klick auf den 'On/Off' Button ein- und auch wieder ausgeschaltet werden. Direkt darunter befindet sich der Lautstärkeregler für das Mikrofon.

Im Masterbereich haben wir einmal den Gesamtlautstärkeregler und den die Talkoverfunktion. Mit dem Talkoverregler wird die Lautstärke eingestellt auf die man den Master dann reduzieren möchte bei der Nutzung des Talkover. Der Talkover kann dann mit gedrückthalten des Talkoverbuttons (erster Button im Master) aktiviert werden. Der Zweite Button ist mit dem Mute-Effekt belegt. Der Talkover ist gedach, um die Musik zu dimmen während man das Mikrofon nutzt.

Einstellungen

Unten-rechts befindet sich ein Button der mit einem Klick die Einstellungen öffnet. In den Einstellungen haben wir nun verschiedene Optionen.

Crossfade
Unter Crossfade kann man einstellen wie lange der Crossfader für einen Welchsel von einem Deck auf das andere braucht.

AutoDj Delete
Wird diese Einstellung auf 'Ja' gestellt, entfernt der AutoDJ (bei aktiviertem AutoDJ) den jeweiligen Track aus der Playlist, sobald er diesen in ein Deck einlädt.

Shortcuts
Mix my Web kann nicht nur mit dere Maus gesteuert werden, sondern hat viele Funktionen auf verschiedene Tasten gelegt. Diese können hier aber auch ausgeschaltet werden. Die Belegung der Tasten können im unteren Bereich eingesehen werden.

Uhr anzeigen
Im unteren Bereich des Master wird Standardmäßig eine Uhr mit der aktuellen Uhrzeit angezeigt. Wenn man dies nicht möchte, kann diese heir ausgeschaltet werden.

Broadcast

Mix my Web kann auch für Webradiosendungen verwendet werden. Aus Sicherheitsgründen der Scripttechnologie kann die Ausgabe leider nicht direkt aus Mx my Web an den Server erfolgen. Daher empfehlen wir die Verwendung von 'butt' einer Streamsoftware, welche das Audiosignal von Mix my Web an den Server sendet.
Nach der Installation starte 'butt'. Dann klicke rechts auf 'Settings', um die Einstellungen zu öffnen. Unter dem Tab 'Main' findest du die Server ('Serversettings'). Dort klicke unter 'Server' auf den Button 'ADD' und gib dort deine Serverdaten ein und speicher sie dann. Als nächstes kannst du mit einem Klick auf den Butten 'ADD' unter 'Stream Infos' Informationen zur Sendung eingeben. Gehe nun auf den Tab 'Audio' und wähle bei 'Audio Device' die Option 'Stereomix' (ab Windows Vista muss dieser erst über Aufnahmegeräte aktiviert werden). Dann schließe wieder die Einstellungen und klicke im Hauptfenster von 'butt' auf den 'Play' Button. Schon sollte in dem blauen Fenster die Stream time stehen, sofern die eingegebenen Serverdaten korrekt waren, und du bist auf Sendung.

Jedoch kann man mit Mix my Web den aktuellen Titel an den Streamserver übergeben. Dazu öffne die Einstellungen durch einen Klick auf den Button unten-rechts.
Unter dem Bereich Broadcast können nun die erforderlichen Daten eingegeben werden.
Sobald 'Metadaten senden' auf 'Ja' gestellt ist, sendet Mix my Web den aktuellen Titel an den Server.

Sicherheit und Statistiken

Sicherheit ist gerde in der heutigen Zeit immer ein großes Thema. Nun stellt sich auch hier sicherlich die Frage, was wir alles an Daten bekommen und speichern von den einzelnen Nutzern.
Da auch wir schon immer für Sicheheit stehen, möchten wir an dieser Stelle versichern, dass alle eingegebenen Daten und Informationen wie auch die Nutzung von Mix my Web keine Auswirkung auf die Privatsphäre haben. Es werden lediglich Informationen für die Anmeldung und allgemeine Nutzerstatistiken von uns gesammelt und gespeichert. Wir sind in einer Form an die in Mix my Web genutzen Inhalte, wie die einzelnen Titel interessiert und es werden dazu auch keine Informationen an unsere Server gesendet. Daher haben wir keine Einsicht wie genau der einzelne User Mix my Web nutzt.
